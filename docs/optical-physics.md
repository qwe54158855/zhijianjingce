# 光学物理约束注入方案 — 技术详解

> **用算法代偿硬件** — 将 Rayleigh/Mie 散射理论编码为可微神经网络
> 实现 266nm→193nm 虚拟波长转换 + 15°-75° 多视角融合检测

---

## 1. 为什么需要光学物理？

**v1.0 的四个问题**：纯数据驱动的 CycleGAN 黑箱映射存在物理不合理、不可解释、波长融合浅层、单视角盲区。

**核心洞察**：设备只有 266nm DUV 光源（常见配置），让 AI 虚拟生成 193nm 光照效果。

| 物理量 | 266nm | 193nm | 增益 |
|--------|-------|-------|------|
| Rayleigh 散射强度 | 基准 1.0× | **(266/193)⁴ ≈ 3.5×** | 小缺陷增强 3.5 倍 |
| Abbe 衍射极限 | ~133nm | **~97nm** | 分辨率提升 27% |
| SiC 穿透深度 | ~0.1μm | ~0.05μm | 更极端表面敏感 |

> 不买昂贵的 193nm 深紫外光源（单套 >50 万，对华出口管制），**用算法换硬件**。

## 2. 散射理论基础

### Rayleigh 散射（小缺陷：d << λ）
`I_scat ∝ (1/λ⁴) × |(n²-1)/(n²+2)|² × d⁶`
- 与波长四次方成反比：266nm→193nm → 3.5×
- 与缺陷体积平方成正比：10nm 颗粒散射是 5nm 的 64 倍
- SiC(n≈2.6) 对比度因子 ~0.67，远高于非晶颗粒(n≈1.5)的 0.04

### Mie 散射（大缺陷：d ~ λ 或 d > λ）
Maxwell 方程组精确解析解，512 点 LUT 预计算，`F.grid_sample` 可微插值。

### Rayleigh-Mie 混合模型
```
gain(s) = w_r(s) × 3.5 + (1-w_r(s)) × Mie_LUT(s)
s∈[0,1], w_r(s)=exp(-s²/2σ²), σ=0.3
s→0: Rayleigh 主导 → 3.5×
s→1: Mie 主导 → 1.2-2.0×
```
光谱角约束：`clamp[1.0, 8.0]`

### 各向异性散射
| 缺陷类型 | 散射特性 | 垂直/平行差异 |
|---------|---------|-------------|
| Particle 颗粒 | 各向同性 | ~1× |
| Scratch 划痕 | 高度各向异性 | 10-20× |
| Chipping 崩边 | 介于两者之间 | 3-5× |

## 3. 极坐标变换与方位角物理

**为什么极坐标**：晶圆是圆形薄片，边缘环形区域仅占 512×512 的 35-40%。

**关键同构**：方位角旋转 = 特征图水平平移
```
Δx = round(Δθ × 512 / 360)   # 像素偏移
torch.roll(feature, shifts=shift, dims=-1)  # 零参数零计算
```

**13 目标角度**：15°, 20°, 25°, 30°, 35°, 40°, 45°, 50°, 55°, 60°, 65°, 70°, 75°（步长 5°）

## 4. PISM — 可微散射物理模块

**位置**：共享编码器之后、检测分支之前
**设计哲学**：**物理 90 + 学习 10** — 公式保证下限，学习逼近上限

### 三步计算流程

**① 局部尺度估计**（~10K 参数）
每像素估计散射尺度 s∈[0,1]（小缺陷 s→0，大缺陷 s→1）
`Conv2d(C→16) → BN → ReLU → Conv2d(16→1) → BN → Sigmoid`

**② Rayleigh-Mie 混合增益计算**
```python
w_r = torch.exp(-s² / (2·0.3²))
combined_gain = w_r · 3.5 + (1-w_r) · Mie_LUT(s)
combined_gain = combined_gain.clamp(1.0, 8.0)
```

**③ 物理增强 + 残差补偿**（0.1 缩放因子确保物理驱动主导）
```python
feat_193 = feat_266 · combined_gain + 0.1 · residual_net(feat_266)
```

| 维度 | 纯数据驱动 | PISM 物理先验 |
|------|-----------|-------------|
| 未见缺陷类型 | 可能乱猜 | 物理公式保底 |
| 数据量需求 | 大量标注 | 90% 来自物理，仅需少量校准 |
| 可解释性 | 黑箱 | 输出 gain_map/scale_map 诊断 |
| 参数效率 | 大网络自己摸索 | 仅 0.16M 参数 |

## 5. ASG — 多方位角度散射生成器

**一句话**：把晶圆从圆形展开为长条形，旋转 = 水平滑动（`torch.roll` 零成本）。

### 三层结构
1. **几何重投影**（0 参数）：`shifted = torch.roll(feat, shift=θ×512/360, dims=-1)`
2. **散射强度调制**（~13K）：`feat = shifted × (1 + 0.3 × Tanh(angle_enc + scale))`
3. **轻量补全**（~10K, 仅 F1）：`feat = feat + 0.1 × completion_net(feat)`

| 指标 | 值 |
|------|-----|
| 13 角度总参数量 | ~23K |
| 生成速度 | < 0.1ms |
| 对比 GAN/NeRF | 节省 1000+ 倍参数和推理时间 |
| 小缺陷召回率提升 | +5-8% |

## 6. 双光路并行检测

### 为什么不能共享权重？
两个波长的散射特性差异太大，共享参数会导致两头都不靠的折中。

- **266nm 分支**（全通道基准）：F2(112→56), F3(224→112), F4(448→224) — 定位精确
- **虚拟 193nm 分支**（通道裁剪）：F2 全通道保留（3.5× 对小缺陷最有利），F3/F4 裁剪 43% — 召回率高

**batch=13 优化**：一次前向替代 13 次循环（~4ms vs ~260ms）

**联合 NMS**：266nm 定位（高精度）+ 193nm 召回（高灵敏度）= 最优结果

## 7. 两级融合策略

### 角度注意力融合（13→1）
```python
W_final = 0.7 × W_physics + 0.3 × W_learned
```
- W_physics：ASG 自评估置信度归一化
- W_learned：Conv2d(13C→64)→ReLU→Conv2d(64→13)→Softmax

### SGF 散射引导融合（波长融合）
```
w_193 = Sigmoid(Conv2D(gain_map ‖ scale_map))   # ~8K 参数
fused = w_266 × pred_266 + w_193 × pred_193
```
- gain_map↑ → w_193↑（高增益区域信赖 193nm）
- s→0(Rayleigh) → w_193 趋近 0.9
- s→1(Mie) → w_193 趋近 0.4-0.5

---

## 8. 物理约束损失函数

**散射一致性损失 L_scat**（λ=0.05）：
`L_scat = Σ ||feat_266 - PISM(feat_193)||²`（torch.no_grad 防止梯度回传）

**光谱角损失 L_spec**（λ=0.02）：
`L_spec = mean[ReLU(1.0 - ratio) + ReLU(ratio - 8.0)]`，ratio = |feat_193|/|feat_266|
