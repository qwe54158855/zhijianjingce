# 光电设计竞赛 — 技术部分修改指导

> **来源**：光学物理约束注入方案 PPT（`wafer-optical-physics_20260717_164223.pptx`）
> **目标**：第十四届全国大学生光电设计竞赛创意组策划书
> **架构**：一骨干双分支 + 光学物理约束注入 v2.0（PISM + ASG + 双光路 + SGF）

---

## 核心架构全景

```
暗场图 → 在线校准 → RepViT-M0.9 编码器 → 增强解码(明场图) + 266nm/193nm 双光路检测
                                                                    ↓
                                                    ASG 13角度 + PISM + SGF → 联合 NMS → 缺陷列表
```

**关键修正**（基于 v2.0 的技术路线更新）：
1. PISM 增益：Rayleigh-Mie 混合模型（不是纯常数 3.5）
2. ASG：13 角度 15°-75° + torch.roll 零成本
3. 193nm 分支通道裁剪（F2 全通道保留，F3/F4 43%）
4. SGF 融合：物理增益感知动态权重
