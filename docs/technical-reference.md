# 半导体晶圆边缘缺陷检测一体化模型 — 技术参考

> **版本**：v2.0（压缩版） | **场景**：SiC/GaN 第三代半导体边缘检测、28nm 制程快速抽检、中小晶圆厂模块化部署
> **核心约束**：模型 < 10M 参数、INT8 < 8MB、ARM Cortex-A76 单核 < 80ms/帧、12 寸晶圆总耗时 < 3min
> **核心创新**：光学物理约束注入 v2.0（PISM + ASG + 双光路）

---

## 1. 项目背景与三大核心难题

### 三大难题

| 难题 | 量化 | 应对方案 |
|------|------|---------|
| ① 国产光源暗场 SNR 不足 | 暗场 15-20dB vs 明场 30dB+ | 学习型暗场→明场域迁移 |
| ② ARM 算力严重受限 | 单帧 < 80ms，总耗时 < 3min | RepViT 轻量骨干 + INT8 量化 |
| ③ 第三代半导体样本稀缺 | 仅 30-50 张标注 | 三阶段半监督训练（50→10000+ 等效样本） |

## 2. 总体技术架构 — 一骨干双分支

```
暗场图 (512×512) → 共享编码器 RepViT-M0.9 → 增强解码分支(→明场图) + 检测输出分支(→缺陷框)
```

**优化目标**：`min L_total = λ_e·L_e(D_e(E(x)), y_e) + λ_d·L_d(D_d(E(x)), y_d)`

**参数预算**：
| 组件 | 参数量 | 占比 |
|------|--------|------|
| 共享编码器 | ~5.1M | 68% |
| 增强解码 | ~1.2M | 16% |
| 检测输出 | ~1.2M | 16% |
| 合计(~7.5M) + 物理模块(~0.31M) | **~8M** | 余量 18.7% |

---

## 3. 共享编码器 — RepViT 轻量骨干

**RepViTBlock**：token_mixer（DWConv3×3 + SE 注意力）+ channel_mixer（倒残差 + 残差连接）

**RepVGGDW 重参数化**（核心）：训练时三分支，推理前融合为单路 3×3 DWConv

**多尺度特征**（4 层输出）：
| 层级 | Stride | 分辨率 | 通道 | 用途 |
|------|--------|--------|------|------|
| F1 | 4 | 128×128 | 56 | 增强（纹理细节） |
| F2 | 8 | 64×64 | 112 | 增强 + 小目标检测 |
| F3 | 16 | 32×32 | 224 | 检测（中层语义） |
| F4 | 32 | 16×16 | 448 | 检测（高层语义） |

**为什么 RepViT**：原生重参数化 + JIT 兼容 + ARM NEON 友好。Top-1 78.7%，ARM 延迟 ~0.9ms。

---

## 4. 增强解码分支 — 暗场→明场域迁移与降噪

**架构**：
```
F1/F2/F3 → 通道对齐 → 自底向上融合(MSEFBlock×2) → Denoiser 降噪 → Conv1+ Sigmoid → 增强明场图
```

**MSEFBlock**：`LayerNorm → DWConv3×3 → SE 注意力 → 乘积融合 → 残差连接`

**Denoiser 降噪**：U 型结构 + 瓶颈注意力（MHSA 精度优先 / DWConv7×7 速度优先，运行时切换）

**四种抗混淆机制**：
1. 全局上下文背景参考（瓶颈注意力区分噪声 vs 缺陷）
2. 残差学习（宁漏勿杀）
3. 多尺度交叉验证（F1+F2 双尺度接入）
4. LPIPS 感知损失监督

---

## 5. 检测输出分支 — Anchor-free 缺陷检测

**四项精简优化**：
1. 去掉独立 Neck（F2/F3/F4 天然特征金字塔）
2. C2f_MSD 替代标准 C2f（多尺度深度可分离）
3. 新增 F2 小目标层（stride=8，10px 缺陷可分辨）
4. 简化回归头（去 DFL，参数量减少 93%）

**缺陷类别**：
| ID | 缺陷 | 典型尺寸 | 检测难度 |
|----|------|---------|---------|
| 0 | Chipping 崩边 | 10-200μm | 中 |
| 1 | Particle 颗粒 | 1-50μm | 高 |
| 2 | Scratch 划痕 | 50-500μm | 低 |
| 3 | Dislocation 位错 | 5-100μm | 高 |

## 6. 光学物理约束注入 v2.0

### 核心洞察：用算法代偿硬件

设备只有 266nm DUV 光源，AI 虚拟生成 193nm 光照效果。Rayleigh 散射增益: **(266/193)⁴ ≈ 3.5×**。

### 五大模块

| 模块 | 参数 | 功能 |
|------|------|------|
| **PISM** | 0.16M | 可微散射物理，266nm→193nm 虚拟转换 |
| **ASG** | 0.01M | 多方位角度散射生成（13 角度，torch.roll 零成本） |
| **193nm 检测** | 0.12M | 通道裁剪双光路检测 |
| **SGF + 角度融合** | 0.008M | 散射引导融合 + 角度注意力 |
| **物理约束损失** | — | L_scat + L_spec |

### PISM 三步流程
1. **局部尺度估计**：每像素散射尺度 s∈[0,1]
2. **Rayleigh-Mie 混合增益**：`gain = w_r·3.5 + (1-w_r)·Mie_LUT(s)`，clamp[1.0, 8.0]
3. **物理增强 + 残差补偿**：`feat_193 = feat_266·gain + 0.1·residual_net(feat_266)`

**设计哲学**：物理 90 + 学习 10 —— 公式保下限，学习逼近上限。

### ASG 多角度生成
极坐标展开后，方位角旋转 = 特征图水平平移（`torch.roll` 零参数零计算）。
13 角度：15°-75°，步长 5°。

### 双光路并行检测
- **266nm 分支**（全通道）：定位精度高
- **193nm 分支**（通道裁剪）：召回率高（F2 全通道保留，3.5× 增益对小缺陷最有利）
- batch=13 一次前向（替代 13 次循环，~4ms vs ~260ms）

### 两级融合
- **角度注意力**：`0.7×W_physics + 0.3×W_learned`（13→1）
- **SGF 波长融合**：`w_193 = Sigmoid(Conv(gain_map ‖ scale_map))`，~8K 参数

### 损失函数
- 散射一致性 L_scat（λ=0.05）：约束虚拟 193nm 散射响应
- 光谱角 L_spec（λ=0.02）：约束增益 1.0≤ratio≤8.0

## 7. 三阶段半监督训练策略

```
阶段一：CycleGAN 无监督预训练（0 标注，100 epoch，FID<30）
  → 阶段二：少量样本联合微调（50 张标注 + Copy-Paste/Mosaic/MixUp → 10000+ 等效样本，200 epoch）
    → 阶段三：知识蒸馏压缩（7.5M→4.2M，KL+特征蒸馏，50 epoch，精度损失<1%）
```

| 损失项 | 阶段一 | 阶段二 | 阶段三 |
|--------|--------|--------|--------|
| CycleGAN (L1+GAN+LPIPS) | 10+1+0.06 | — | — |
| 物理约束 (L_scat+L_spec) | 0.05+0.02 | 0.05+0.02 | — |
| 检测 (266+193+fused) | — | 1+1+0.5 | — |
| 蒸馏 (KL+特征) | — | — | 0.9+0.1 |

**分级冻结**：Epoch 0-50 训练高层+检测头 → 51-100 解冻 stage2 → 101-200 全部解冻 lr=1e-5

## 8. 模型压缩与 TorchScript JIT 导出

**融合顺序**：Conv2d_BN → RepVGGDW → Residual → Dropout（Conv2d 节点减少 40-50%）

**INT8 静态量化**：per-channel 权重 + per-tensor 激活（qnnpack 后端），敏感层保留 FP32
- 校准：20 张代表性晶圆图
- 精度：增强 MaxDiff<1e-2，检测 mAP 下降<0.5%

**JIT 校验**：
```
FP32 MaxDiff < 1e-5 | INT8 MaxDiff < 1e-2 | 类别一致性 > 99% | 文件 < 8MB(M0.9) / < 3MB(M0.6)
```

## 10. 晶圆场景专属优化

**极坐标边缘展开**：512×512 → 512×128，计算量减少 60%，ARM 推理 +30%
（晶圆中心区域不适用，`enable_polar` 开关控制）

**在线自适应校准**：启动时收集 5-10 张正常晶圆灰度统计量，补偿光源老化
（6 个月未校准误检率上升 +12% vs 校准 +2%）

**参数化配置系统**：JSON 配置文件适配不同工艺条件（灵敏度/晶圆尺寸/输出格式）

## 12. 风险应对矩阵

| 风险 | 概率 | 应对 |
|------|------|------|
| INT8 量化 mAP 降 > 1% | 中 | 敏感层 FP32 / QAT 5-10 epoch |
| ARM 达不到 80ms | 低 | 角度 13→7 / 关 ASG / 极坐标 / M0.6 |
| 小缺陷召回提升不达 5% | 中 | 检查 PISM 增益 / 增大 L_scat 权重 |
| 产线光源漂移 | 低 | 在线自适应校准 + 季度微调 |

**降级预案**（配置文件单行切换）：
```
全功能(13角度×2波长) → 7角度 → 仅双波长 → 原始单分支 → M0.6 极致剪裁
```

## 13. 端到端流水线与延迟分解

```
266nm 输入 → 在线校准 → 极坐标展开(可选) → RepViT 编码器(~15ms)
  → 266nm 检测(~3ms) + ASG 角度生成(~2ms) + PISM 物理(~1ms) + 193nm batch 检测(~4ms)
  → 角度融合+SGF(~1ms) + 增强解码(~4ms) → 联合 NMS → 缺陷列表
```
ARM 估算总量：**~30ms/帧**（全功能），留有 50ms+ 余量
